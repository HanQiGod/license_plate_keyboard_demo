# license_plate_keyboard_demo

Flutter 自定义车牌号键盘示例项目。

## 功能说明

- 支持普通车牌 `7` 位输入
- 支持新能源车牌 `8` 位输入
- 首位自动显示省份简称键盘
- 第二位自动显示字母键盘
- 后续位显示字母数字键盘
- 支持删除、收起键盘

主页面文件：`lib/main.dart`

## 目录结构

```text
lib/
  main.dart
  src/
    app.dart
    pages/
      plate_keyboard_page.dart
    widgets/
```
